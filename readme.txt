Box position ( left, right, top, bottom ),1. Slider action ( hover, click ),1. Open / close time ( in ms )1. Start / open / close opacity of SN Google Plus box1. Chose 1 of 12 unique icons.**SN Google Plus** require jQuery 1.4+== Installation == Copy the directory '**SN Google Plus**' to your WordPress '*/wp-content/plugins*' folder.1. Activate the '*SN Google Plus*' plugin in your WordPress admin '*Plugins*'2. There is all.== Screenshots ==1. 'Plugin' screenshot. 2.
